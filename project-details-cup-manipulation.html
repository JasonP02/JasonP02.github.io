Autonomous Cup Manipulation

Research project using LoCoBot for precise liquid handling

Overview

Two-semester research project focused on developing autonomous manipulation capabilities for handling water-filled cups using the LoCoBot platform. The project combined computer vision, motion planning, and precise control systems.

Technical Details

• ROS implementation for robot control
• Computer vision for cup detection and tracking
• Motion planning algorithms
• Custom end-effector design
• Integration with LoCoBot platform

ROS Python Linux Computer Vision Motion Planning

Key Learnings

• Strong foundation in Linux and ROS
• Dependency management in complex systems
• Core software engineering principles
• Research methodology and documentation

Challenges & Solutions

The primary challenge was achieving reliable liquid handling without spills. This was addressed through careful trajectory planning and the development of specialized control algorithms that accounted for liquid dynamics.
